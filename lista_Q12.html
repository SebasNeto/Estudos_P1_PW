Aqui está um exemplo de uma página HTML que exibe uma caixa de alerta com o texto "Botão Presionado" quando o usuário clica no botão. Utilizaremos o método addEventListener para programar o evento de clique no botão. Exemplo de Botão com Alerta Este código cria uma página HTML simples com um botão que, quando clicado, exibe uma caixa de alerta usando a função window.alert. O método addEventListener é utilizado para programar o evento de clique no botão, garantindo uma abordagem moderna e recomendada para a manipulação de eventos em JavaScript.
