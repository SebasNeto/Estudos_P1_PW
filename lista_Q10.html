O código fornecido cria uma interface simples com dois botões que permitem adicionar e remover estilos a um elemento
com o ID icomp. Resumo do Funcionamento Quando o botão "Adicionar Estilos" é clicado: O
com o ID icomp recebe um fundo cinza claro (lightgray), texto vermelho (red) e tamanho de fonte de 32 pixels (32px). Quando o botão "Limpar Estilo" é clicado: Os estilos aplicados ao
icomp são removidos, retornando ao estilo padrão do navegador ou aos estilos definidos em outra parte do CSS. Example
