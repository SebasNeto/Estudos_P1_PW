
A tag <meta charset='UTF-8'> define a codificação de caracteres usada na página, 
o que é essencial para a correta exibição de caracteres especiais. 

A especificação HTML5 requer que esta tag esteja dentro do elemento <head> para ser corretamente reconhecida pelo navegador.
